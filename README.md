# Venda de Games
Projeto acadêmico em C# Windows Forms da matéria de Técnicas de Programação com CRUD completo com tela de login, tela de menu, cadastro de usuários, cadastro de jogos e cadastro de vendas. Projeto com o objetivo de aprender a conexão com banco de dados **MySQL** e login com usuários diferentes.
